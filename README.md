# Kanban Board

A web-based Kanban Board application designed to manage tasks efficiently across different stages of a workflow.

## Live Demo

https://your-kanban-app.onrender.com

## Repository

https://github.com/maahi-arya/kanban-board

## Tech Stack

* HTML
* CSS
* JavaScript

## Key Features

* Task creation, editing, and deletion
* Workflow management (To Do, In Progress, Done)
* Drag-and-drop functionality (if implemented)
* Clean and responsive user interface

## Project Description

This project demonstrates the implementation of a task management system based on the Kanban methodology. It focuses on building an intuitive user experience and handling dynamic UI interactions efficiently.

## Run Locally

Clone the repository and open `index.html` in your browser.
